本篇文章将解释一些较为小众与专业的用语，旨在帮助有需要或对此感兴趣的读者进一步了解部分标签的含义，同时更好地使用本站的标签分类系统。

## TVA

**Television Animation，电视动画**

指专为电视播出而制作的动画作品。即使动画电影或 OVA 等作品后来在电视上播放，也不属于 TVA 分类。

## AM

**Animation Movie，动画电影 / 剧场版动画**

指以电影院（剧场）为主要首发渠道的动画作品，通常拥有更高的制作成本与更长的放映时长。

## ONA

**Original Net Anime，网络动画 / Web 动画 / 原创网络动画**

指最初通过互联网渠道进行发布的动画作品，是随着网络发展而兴起的新形式。

随着流媒体的发展，出现了只能在网络平台上看的商业动画，这些类似于电视动画的商业动画是否能被称作网络动画还有待商议。 

## OVA

**Original Video Animation，原始视频动画**

指不通过电视或影院，而是以实体光盘等媒介为首发渠道的动画作品。内容上常作为 TVA 的外传、续篇或是一些更具实验性的独立企划。

## Anime

**アニメ，日本动画 / 日式动画**

广义上指日本出品的动画作品，涵盖 TVA、AM、ONA、OVA 等所有形式。通常以电视播出为主要渠道，是国际上对日本动画的通用称呼。

## Manga

**マンガ，日本漫画**

指日本漫画作品，或具有日本漫画风格的漫画创作。

## Galgame

**ギャルゲーム，美少女游戏**

一种以与女性角色互动及发展情感关系为主要玩法的电子游戏形式，常见表现方式为视觉小说。以角色塑造与剧情体验为核心。

## Astro

Astro 是最适合构建像博客、营销网站、电子商务网站这样的以内容驱动的网站的 Web 框架。Astro 以开创了一种新的前端架构而闻名，与其他框架相比它减少了 JavaScript 的开销和复杂性。[^1]

## Hugo

Hugo 是由 Go 编写的快速现代静态网站生成器，以其极快的构建速度而闻名。Hugo 旨在为网站最终用户提供最佳的查看体验，并为网站作者提供理想的编写体验。[^2]

## VitePress

VitePress 是一个静态站点生成器，专为构建快速、以内容为中心的站点而设计。简而言之，VitePress 获取用 Markdown 编写的内容，对其应用主题，并生成可以轻松部署到任何地方的静态 HTML 页面。[^3]
